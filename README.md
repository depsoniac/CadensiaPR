# Cadencia - GitHub Pages Inline

Esta versión está hecha para evitar problemas de rutas en GitHub Pages.

Las imágenes y GIFs están incrustados directamente dentro de `index.html`, así que no depende de carpetas `assets/`.

## Cómo subirlo

1. Descomprime este ZIP.
2. Sube a GitHub solo estos archivos/carpetas:
   - `index.html`
   - `.github/`
   - `.nojekyll`
   - `README.md`
3. En GitHub ve a `Settings → Pages`.
4. En `Build and deployment`, selecciona `GitHub Actions`.
5. Corre el deploy desde `Actions`.

No subas el ZIP completo al repo.
